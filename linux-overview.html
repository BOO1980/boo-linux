Linux Overview

  • Linux has a graphical user interface and it works pretty much like the GUIs on other systems that you afre familiar with
  • A command line, or terminal, is a text based interface to the system
  • You are able to enter commands on the command line by typing them on the keyboard and feedback will be give to you similary as text
  • The command line typically presents you with a prompt
  • user@bash is a prompt, after which we enter a command
  • typically a command is the first thing you type (ls)
  • After the command we type in the arguments (-/home/billingham)
  • The first command line argument (-l) is also referred to as an option
  • Options are typically used to modify the behaviour of the command
  • Options are usually listed before other arguments and typically start with a dash
  • Most commands produce output and it will be listed straight under the issuing of the command
  • After the command has run and the terminal is ready for you to enter another command prompt will be displayed
  • To open a terminal in Windows use putty
